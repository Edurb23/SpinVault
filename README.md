# SpinVault

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 14.0.0.

## About the project

SpinVault is a study project created to practice Angular concepts and build a vinyl record collection interface. The application allows the user to explore and manage vinyl records in a simple and visual way.

## How to run the project

Before starting the Angular application, make sure to run the backend first.

### 1. Start the backend

Open a terminal and run:

cd backend
npm run start

### 2. Start the Angular frontend

In another terminal, run:

ng serve

Then navigate to:

http://localhost:4200/

The application will automatically reload if you change any of the source files.

## Development server
